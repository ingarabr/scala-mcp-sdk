# Contributing

We welcome contributions that fit the library's scope.

## Scope

This is a **protocol library**, not a framework. We focus on:

- MCP protocol implementation
- Type-safe server APIs
- Transport abstractions

We intentionally leave out:

- HTTP server management
- Logging infrastructure
- Configuration management
- Metrics/observability

## Questions?

Open an issue on [GitHub](https://github.com/ingarabr/scala-mcp-sdk/issues).
